# SOC Lab Open-Source


Laboratório prático de Segurança Defensiva simulando um SOC real, com arquitetura modular, integração entre ferramentas open source e foco total em detecção, resposta e investigação de incidentes.

Este repositório não é apenas teórico. Ele documenta a construção e a operação de um mini‑SOC funcional, inspirado em ambientes corporativos reais.

## Objetivo do Repositório

### Demonstrar conhecimento prático em:

- Arquitetura SIEM/SOAR realista

- Integração entre ferramentas open-source

- Automação de resposta a incidentes

- Investigação e DFIR

- Uso de IA local aplicada à segurança

> Projeto focado em prática real, automação e resposta a incidentes.
